To host your website for free, you can use platforms like GitHub Pages, Netlify, or Vercel. I'll provide you with steps for each of these platforms:

1. GitHub Pages

Steps:

  1. Create a GitHub Account: If you don't have one, sign up at GitHub.

  2. Create a Repository:

    • Go to your GitHub profile and click on "Repositories".
    • Click the "New" button to create a new repository.
    • Name your repository (e.g., recipez-by-sufi), and make sure to check the "Public" option.
    • Click "Create repository".

  3. Upload Your Files:

    • On the repository page, click "Add file" > "Upload files".
    • Upload your index.html and styles.css files.
    • Commit your changes.

  4. Enable GitHub Pages:

    • Go to the "Settings" tab of your repository.
    • Scroll down to the "Pages" section.
    • Under "Source", select the branch as main (or master) and the root directory.
    • Click "Save".
    • Your site will be available at https://yourusername.github.io/repositoryname/.

2. Netlify

Steps:

  1. Create a Netlify Account: Sign up at Netlify.

  2. New Site from Git:

    • After logging in, click on "Add new site" > "Import an existing project".
    • Connect to your GitHub account and select the repository you created.
    • Netlify will automatically detect the index.html file and deploy your site.

  3. Deploy Settings:

    • Follow the prompts to configure and deploy your site.
    • Your site will be available at a Netlify subdomain, which you can customize later.

3. Vercel

Steps:

  1. Create a Vercel Account: Sign up at Vercel.

  2. New Project:

    • After logging in, click on "New Project".
    • Connect to your GitHub account and select the repository you created.

  3. Deploy:

    • Follow the prompts to configure and deploy your site.
    • Vercel will deploy your site to a subdomain, which you can customize.

Summary

  • GitHub Pages: Good for simple static sites, integrated with GitHub.
  • Netlify: Easy to use, good for both static and dynamic sites, includes continuous deployment.
  • Vercel: Similar to Netlify, good for static and serverless functions, includes continuous deployment.

Choose the one that suits you best and follow the steps to get your website online. If you encounter any issues or need further assistance, feel free to ask!
